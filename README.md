# Dashboard de Análisis Empresarial — Ventas y Rendimiento

## Descripción
Proyecto de análisis de datos de una empresa comercial que combina 
análisis de ventas, rendimiento de empleados e inventario para 
apoyar la toma de decisiones gerenciales.

## Tecnologías utilizadas
- Python 3.12
- pandas
- matplotlib / seaborn
- MySQL
- SQLAlchemy

## Estructura del proyecto
- entrada/ → datos fuente
- salida/ → reportes y gráficas generadas
- scripts/ → archivos Python

## Análisis realizados
- Tendencia de ventas 2022-2024
- Rendimiento de empleados por departamento
- Análisis salarial por nivel y departamento
- Proyección de ventas

## Capturas del Análisis

### Ventas por Categoría y Región
El análisis revela que Electrónica es la categoría más importante 
representando el mayor volumen de ventas. La región Centro lidera 
en todas las categorías.

### Proyección 2025
Se proyecta un crecimiento del 15% basado en factores estacionales
y tendencias históricas 2022-2024.

# Dashboard de Análisis Empresarial — Ventas y Rendimiento

## Descripción
Proyecto de análisis de datos de una empresa comercial que combina 
análisis de ventas, rendimiento de empleados e inventario para 
apoyar la toma de decisiones gerenciales.

## Tecnologías utilizadas
- Python 3.12
- pandas
- matplotlib / seaborn
- MySQL
- SQLAlchemy

## Estructura del proyecto
- entrada/ → datos fuente
- salida/ → reportes y gráficas generadas
- scripts/ → archivos Python

## Análisis realizados
- Tendencia de ventas 2022-2024
- Rendimiento de empleados por departamento
- Análisis salarial por nivel y departamento
- Proyección de ventas
